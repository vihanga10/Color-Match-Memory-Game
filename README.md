Colour Match Memory Game

Welcome to the Color Match Memory Game repository! This is a fun and challenging memory game project where players try to match colors hidden behind tiles. The game is designed to improve memory skills and provide an engaging experience for users of all ages.

About the Project

The Color Match Memory Game is a work-in-progress game application developed using SwiftUI. The game features a grid of tiles that players tap to reveal colors underneath, aiming to find matching pairs. The game includes features such as:

	•	Time attack mode to challenge your speed
	•	Score tracking and move counts
	•	Intuitive and colourful UI design
	•	Smooth animations for tile flipping and matching
	•	Increasing difficulty levels as you progress

Currently, the game is under active development, with new features, bug fixes, and UI improvements being added regularly.

Features

	•	Tile matching gameplay: Flip tiles to find matching colours.
	•	Score and move tracking: Keep track of your progress and performance.
	•	Timer mode: Challenge yourself against the clock.
	•	Animations: Smooth and fun tile flip animations.
	•	User-friendly interface: Designed with simplicity and accessibility in mind.
	•	Multiplayer mode
	•	Power-ups
	•	Enhanced UI themes

How to Run

Clone this repository:

git clone https://github.com/vihanga10/Color-Match-Memory-Game.git


Open the project in Xcode:

	•	Navigate to the project folder.
	•	Open the .xcodeproj or .xcworkspace file.
	•	Build and run the project on a simulator or physical device.
